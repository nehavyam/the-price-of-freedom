# The Price of Freedom 

An immersive, choice-driven interactive text adventure game built entirely with vanilla web technologies. Styled like a classified government case file, the game challenges players to make critical decisions to escape captivity from Cell Block C.

**[▶ Play the Game Live Here!](https://nehavyam.github.io/the-price-of-freedom/)**

---

## Gameplay & Premise

You have been locked away for four years for speaking out against the government. Tonight, a lapse in guard security presents a fleeting window of opportunity.

Every choice is logged. Every escape has a cost. Will you make it out alive — or will you be recaptured?

### Features:
- **Dynamic Storytelling:** A branching narrative driven entirely by user choice.
- **Immersive Visuals:** A sleek, retro-modern UI designed to look like an authentic classified dossier.
- **Lightweight & Fast:** Zero external frameworks or dependencies — built purely with vanilla web languages.

---

##  Built With

- **HTML5** — Semantic structure for the narrative layout.
- **CSS3** — Custom styling, typography, and an atmospheric dark-mode presentation.
- **JavaScript (ES6)** — State management, story progression logic,choosing option form the given options .

---

##  How to Run Locally

1. Clone this repository:
```bash
   git clone https://github.com/nehavyam/the-price-of-freedom.git
```
2. Navigate into the project folder:
```bash
   cd the-price-of-freedom
```
3. Open `index.html` in your browser — no build steps or dependencies required.
